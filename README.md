# Workflow Builder UI
## Overview
A simple visual workflow builder built using React functional components and hooks.
Allows users to create workflows using Action, Branch (Condition), and End nodes.
Designed to demonstrate recursive rendering and state management.

## Features
- Add, edit, and delete workflow nodes.
- Branch nodes automatically create True / False paths.
- Workflow flow is maintained when nodes are deleted.

## How to Run
 ```
npm install
npm run dev
```
